# 水利监督检查

面向 iPhone 的现场检查通报 PWA。首版覆盖：

- 抖音式顶部搜索、底部固定菜单和中间“＋”新建检查入口；
- 首页照片按工程满屏显示，上下滑动切换，其他功能页统一白底黑字；
- 一天内添加多个工程，同一工程可连续添加多条问题；
- 工程库保存参建单位资料，新建检查时直接选择并自动带出；
- 点击工程名称可修改工程及参建单位资料；
- 工程名称、参建单位、工程基本情况和问题备注支持语音输入；
- 调用 iPhone 相机拍照，添加简单备注；
- 导入 `.docx` 或 `.txt` 往期通报，在设备内提取历史问题库；
- 导入规范 PDF、Word 或文本，识别条文并在整理问题时自动推荐；
- 根据备注与历史问题库自动生成问题标题、“经查”表述、分类、等级、规范候选和整改要求；
- 人工逐条确认后，按正式模板生成质量/安全监督整改通知书 `.docx`；
- 整改通知书按模板的质量行为、实体工程质量、现场安全隐患等栏目自动归类，问题使用楷体五号；
- 整月检查结束后，汇总全部工程并按2026年第5期模板生成监督检查月通报 `.docx`；
- 月通报自动生成正文统计、县市区及工程编号、问题段落、对应照片和附件1；
- 照片自动跟随对应问题，以双栏方式排版；
- 数据和照片保存于设备本地数据库；
- 安装到 iPhone 主屏幕后，可独立窗口运行并缓存基础界面。

## 本机预览

在本目录启动任意静态文件服务，然后用浏览器访问。部署到 HTTPS 地址后，用 iPhone Safari 打开，选择“分享 → 添加到主屏幕”即可安装。

具体步骤见 [手机安装说明.md](./手机安装说明.md)。

## 首版边界

- Word 导入支持 `.docx`，不支持旧版 `.doc`；
- 历史通报 PDF 导入列入下一阶段；规范库已经支持 PDF；
- 自动推荐的规范条文必须在正式发布前对照规范原文核验；
- 当前自动整理采用本机历史相似度匹配与规则生成，不向外部服务上传照片或通报。
